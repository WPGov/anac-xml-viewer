Software per la visualizzazione di dataset XML su tracciato ANAC (ex AVCP -Legge 190/2012 Art 1.32).

== Description ==

ANAC XML VIEWER è un plugin WordPress pensato per le PA utile alla pubblicazione di bandi di gara ai fini della trasparenza (D.lgs 33/2013) e l'adeguamento normativo richiesto dall’Autorità Nazionale Anticorruzione (specifiche tecniche art. 1 comma 32 Legge n. 190/2012).

https://www.youtube.com/watch?v=cdn082kZogk

= Caratteristiche =
* Visualizzazione tabellare dei bandi di gara a partire da un dataset XML già generato
* Importazione del dataset tramite copia/incolla del contenuto del file o URL.
